Nick Stabile - Literature and Technology Final Project


Welcome to the Frankenstein Word Graph!


This application allows you to select a book, enter one or more words and see a graph of how these words are used throughout the book. Hopefully this tool will provide you with a better understanding of the book and will be useful in developing a critical analysis of the text. While this project was designed for use with Frankenstein, it will work with any text file containing chapters.
